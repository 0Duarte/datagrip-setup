# 🛠️ Setup do DataGrip

## 📚 Janelas Fixas no Menu Superior

Organize a interface do DataGrip com as seguintes janelas fixadas na parte superior:

- **Database Explorer**
- **Services**
- **Structure**
- **Copilot**
- **Files**
- **More Tool Window**

> Isso garante acesso rápido às ferramentas essenciais do ambiente de trabalho.

---
